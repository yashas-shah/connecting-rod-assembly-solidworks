#  Connecting Rod Assembly (SolidWorks)

##  Overview
This project involves the design and assembly of a connecting rod mechanism using SolidWorks.  
The assembly represents a simplified mechanical system used in engines to convert reciprocating motion into rotational motion.

---

##  Components Included
- Connecting Rod (Top Part)
- Connecting Rod (Bottom Part)
- Nuts and Fasteners
- Bushing
- Complete Assembly File
